# Debian-build-server-V2
automated buid scripts for a Debian based distros for server env.

# Usage

1. Clone this repository: git clone https://github.com/eqvaldi/Debian-build-server-V2.git

2. cd Debian-build-server-V2/

3. Run: bash ./create-files.sh

4. Then run: bash ./git-build.sh

5. run: sudo bash ./build-all.sh
